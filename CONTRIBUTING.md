# Contributing to Event Management System

Thank you for your interest in contributing to this project!

## About This Project

This Event Management System API was built by **Akbari Prayag** as a portfolio project demonstrating Django REST Framework skills, including:

- RESTful API design
- JWT authentication
- Custom permissions
- Async task processing
- Comprehensive testing

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/Event-Management-System.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test your changes: `python manage.py test events`
6. Commit your changes: `git commit -m "Add: description of changes"`
7. Push to your fork: `git push origin feature/your-feature-name`
8. Open a Pull Request

## Code Style

- Follow PEP 8 Python style guide
- Use meaningful variable and function names
- Add docstrings to all functions and classes
- Keep functions focused and small
- Write tests for new features

## Testing

Always run tests before submitting:

```bash
python manage.py test events
```

## Reporting Issues

If you find a bug or have a suggestion, please open an issue with:
- Clear description of the problem or feature
- Steps to reproduce (for bugs)
- Expected behavior
- Actual behavior
